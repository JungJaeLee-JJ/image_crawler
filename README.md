이미지 인식과 관련된 딥러닝 프로젝트를 진행하기 위하여, 구글과 네이버에서 키워드를 검색하여 이미지를 크롤링해오는 코드를 작성하였다.
 검색에 필요한 키워드가 엑셀로 저장되어 있는 경우가 많아, 엑셀파일을 파이썬에서 다룰수 있는 openpyxl 라이브러리를 사용하였다. 코드상에는 엑셀의 첫번째 행을 기준으로 검색한다.  
 
 
해당 코드를 수행하기 위해서는 동일 폴더내에 크롬웹드라이버가 설치되어야하며, [여기](https://chromedriver.chromium.org/downloads)에서 다운받을 수 있다.  
라이브러리는 pip install [라이브러리명] 을 통해 다운로드가 가능하다.(단 [환경변수설정](https://m.blog.naver.com/PostView.nhn?blogId=lee95292&logNo=221205091279&proxyReferer=https%3A%2F%2Fwww.google.com%2F)이 되어있어야함. )

# 사용방법  

1. 크롬 웹드라이버와, 필요한 라이브러리를 설치한다.  
2. 파일이름.xlsx로 만들고 검색하고자 하는 키워드들을 첫 번째시트의 첫 번째에 행에 순서대로 작성한다.  
3. 코드내에 엑셀파일경로, 크롬웹드라이버 경로를 설정한다.  
4. 코드를 수행한다.  
유의사항 - 현재 사용중인 크롬 버전에 맞춰서 크롬웹드라이버를 설치하여야 한다.
